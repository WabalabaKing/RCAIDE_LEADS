Installation
============

Requirements
============

- Python 3.x
- Dependencies listed in requirements.txt

Installation Steps
==================

1. Clone the repository:

.. code-block:: bash

    git clone https://github.com/yourusername/RCAIDE_LEADS.git

2. Install dependencies:

.. code-block:: bash

    pip install -r requirements.txt

3. Install RCAIDE_LEADS:

.. code-block:: bash

    pip install -e .
